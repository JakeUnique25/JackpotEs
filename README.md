## Overview

The Jackpot Estimator is a web-based tool for estimating **PCSO lotto jackpots**. It supports the following games:

- **UltraLotto 6/58**  
- **GrandLotto 6/55**  
- **SuperLotto 6/49**  
- **MegaLotto 6/45**  
- **Lotto 6/42**

This estimator considers **minimum guaranteed jackpot amounts** and applies a **snowballing effect** when the total allocation exceeds the minimum.  

---

## Features

- **PCSO branding:** Red, Blue, Yellow, White theme  
- **Official PCSO logo** and **lotto game logos** for clear recognition  
- **Mobile-friendly layout** (iOS & Android)  
- **Dictation-ready input fields** with automatic comma formatting  
- **Minimum jackpot logic & snowballing effect**  
- **Estimated jackpot results** formatted with commas and two decimal places  
- **Standalone web app behavior** via GitHub Pages or browser shortcut  
- **Maker credit** displayed at the bottom  

---

## Usage

1. Open the HTML file or the GitHub Pages link in a browser.  
2. Select a **lottery game** from the dropdown.  
3. Enter **Carry Over Amount** and **Current Draw Allocation** using your keyboard or mobile dictation.  
4. Click **Compute Estimate** to see the estimated jackpot.  

> The estimator automatically applies minimum guaranteed jackpots before snowballing.
